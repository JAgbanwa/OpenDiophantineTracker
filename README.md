# Open Diophantine Tracker

A dependency-free static website for monitoring progress on the open problems in Bogdan Grechuk's
`arXiv:2404.08518` paper, with provenance, filtering, local status overrides, and exportable curator notes.

Open `index.html` in a browser. Local edits are stored in browser `localStorage`; use the Curator section to export
or import that state as JSON.

Seed data reflects `arXiv:2404.08518v8`, last revised 13 Jul 2026.
